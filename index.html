Introdução

Como surgiu?

De acordo com a história fornecida em Schmidhuber, "Deep learning in neural networs: an overview", Neutral Networks (2015), autoencoders foram propostos como um método para o pré-treinamento não-supervisionado in Ballard, "Modular learning in neural networks", Proceedings AAAI (1987).

Definição

Autoencoder é uma rede neural que é treinada para tentar copiar a entrada para a saída.

Onde é utilizado?

  • Remoção de ruído
  • Detecção de anomalia
  • Compressão de dados
